# DOAC Insights

**What 400+ world-class performers agree on** — AI-powered analysis of every Diary of a CEO episode.

Interactive knowledge graph, cross-episode pattern analysis, and distilled key learnings from one of the world's biggest podcasts.

## Features (WIP)
- 🎙️ Full transcript corpus from YouTube
- 🧠 AI-extracted insights per episode
- 🔗 Cross-episode pattern detection
- 📊 Interactive web app with episode links
- 🏷️ Topic clustering (business, health, relationships, psychology, habits, money)
- ⚡ "What do 5+ guests independently agree on?"
- 🤔 Contrarian takes & guest disagreements

## Tech
- yt-dlp for transcript scraping
- Gemini Flash for bulk summarisation
- Embeddings + clustering for pattern detection
- Next.js interactive web app
